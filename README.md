# 📷 Philog — 필름 감성 포토 다이어리

후지필름 카메라로 촬영한 사진을 기록하는 개인 포토 다이어리 웹앱입니다.
Claude AI를 활용하여 기획부터 디자인, 개발, 배포까지 1인으로 구현했으며
현재도 지속적으로 업데이트 중인 프로젝트입니다.

🔗 **배포 URL**: https://philog-686kl4s3l-sueun.vercel.app/

---

## 🛠 기술 스택

| 영역       | 기술                                            |
| ---------- | ----------------------------------------------- |
| Frontend   | React, Next.js 15 (App Router), Tailwind CSS v3 |
| Backend    | Supabase (PostgreSQL, Storage, Auth)            |
| 배포       | Vercel                                          |
| 라이브러리 | exifr.js, fuji-recipes                          |

---

## ✨ 주요 기능

- 📷 사진 업로드 시 EXIF 정보 자동 파싱
- 🎞️ 후지필름 필름 시뮬레이션 / 레시피 자동 인식
- 🔐 회원가입 / 로그인 / 비밀번호 재설정
- 👤 마이페이지 (프로필 설정, 내 사진 모아보기)
- 🏷️ 태그 기능 및 클릭 필터링
- 📱 모바일 반응형 (인스타그램 스타일)
- ♾️ 무한 스크롤 페이지네이션
- 🌙 다크모드 지원
